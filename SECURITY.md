# Security Policy

## Supported Versions

Security fixes are expected to target the latest maintained release of
`event_log`.

Older versions may not receive updates.

## Reporting a Vulnerability

Please do not open a public issue for suspected security vulnerabilities.

Report security issues privately to the maintainers with:

- A description of the issue
- Reproduction steps
- A proof of concept if available
- The affected package version
- Your Windows version and environment details

## Responsible Disclosure

After a report is received, maintainers will investigate the issue, prepare a
fix when necessary, and coordinate disclosure appropriately.
